# Task1 — Карта IT-ландшафта и схема интеграции приложений (AS-IS)

## Артефакты
- `task1-it-landscape-current.drawio` — карта текущего IT-ландшафта банка в разрезе оргструктуры (строки) и бизнес-возможностей 2-го уровня (колонки).
- `task1-application-integration-current.drawio` — схема интеграции приложений с участниками процессов по депозитам и кредитам.

## Что отражено
- Текущий (AS-IS) процесс открытия депозитов и накопительных счетов, включая ручные шаги (`Excel`, `Email`, обработка бэк-офисом).
- Текущий (AS-IS) процесс кредитования, включая пакетные интеграции `1 раз в сутки` между `АБС`, `Кредитным конвейером` и источниками для скоринга.
- Основные участники процессов: клиент, фронт-офис, бэк-офис депозитов, управление кредитных продуктов, кол-центр.

## Принятые допущения
- Для карты IT-ландшафта использованы бизнес-возможности второго уровня, вытекающие из описания кейса (без исходного файла `Business Capability Map`).
- В схеме интеграции выделены типы связей: ручные (`Email/Excel`), API-вызовы и пакетные обмены через БД.
